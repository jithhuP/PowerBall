# 설치 방법 
(추가)+APK 파일로 만들고 파일을 설치할 수 있게 해준다.

[Visual Studio](https://code.visualstudio.com/Download)혹은 [Android Studio](https://developer.android.com/studio)를 다운받아 프로그램을 실행할 수 있다.
